Problemas de Redondeo

Los errores de redondeo ocurren al aproximar números debido a las limitaciones en su representación decimal. Estos errores afectan la precisión y pueden tener consecuencias significativas en áreas como la ingeniería y las finanzas.

Tipos de Errores

  • Error absoluto: Es la diferencia entre el valor real y el valor aproximado, se calcula como: Error Absoluto = |Valor Verdadero - Valor Aproximado|

  • Error relativo: Es el cociente entre el error absoluto y el valor real.



  • Error porcentual: Es el error relativo expresado en porcentaje, se calcula como: Error Porcentual = Error Relativo x 100%
